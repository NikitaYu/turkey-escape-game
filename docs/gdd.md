# Turkey Escape GDD (v1.0 - Nov 18, 2025)

## Overview
Casual top-down browser game: Live turkey escapes hipster tech lab through vents/mazes, dodging hazards, grabbing powerups. Multiverse portals add chaos/upgrades. 90s Diablo grit + Half-Life anomalies + humor.

## Core Loop
Constant forward run + steer/jump. Avoid hazards, collect powerups, beat timer to exit. Non-linear maps, revisit for upgrades/anomalies.

## Mechanics
- Movement: Always running (base 200px/s forward). WASD/arrows steer.
- Jump: Space/tap â€“ hop low walls/gaps. Visual: Sprite enlarge +/or map shrink (test both).
- Hazards: Knives/ovens (instant/full damage), anomalies (half/full).
- Powerups: Speed fast/slow-mo (random, 10s).
- Health: 3 turkey-triangle icons start (fade gray half, outline lost). Expand 5-6 via upgrades.
- Maps: 5 fixed MVP (first 3 tutorial). Later non-linear + anomalies on revisit.

## Visuals
- Style: Monochrome/grayscale B&W comic (black BG #000000, grays #222-CCC, white #FFF).
- Turkey: Prolonged orange triangle (#FF4500).
- Effects: Mild shakes/flashes.

## Audio

- BGM: Hard techno-rock.
- SFX: Damage gobble, collect jingle.

## Story/Humor
Hipster scientists plan "dance feast." Turkey quips/taunts on events.
